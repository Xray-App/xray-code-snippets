## Flow

Depending on the user's request, you may need to obtain different types of data from Xray, such as Test Runs, requirements and coverage information, etc. For each type of data, there are specific instructions to follow, which are described in the sections below. Always follow the instructions for each specific type of data to ensure a consistent and accurate extraction of testing data from Xray.
