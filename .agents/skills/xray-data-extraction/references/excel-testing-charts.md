# Testing charts in Excel

Use `excel-mcp-server` MCP tools to make the Excel files and related charts. 

## Global instructions

- if multiple sheets are required, start by creating sheets first and give the proper names to them, and only after adding the data or charts into them
- always give a proper, short names to sheets; don't use defaults
- always include a relevant title on charts
- for bar charts, include meaningful labels on axes

### Test Runs

- whenever building charts for Test Runs, prefer either bar charts (default) or pie charts.
- always include a sheet for the list of Test Runs, naming it "Test Runs"
