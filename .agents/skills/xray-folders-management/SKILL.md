# Managing Folders in Xray (test repository, test plans)

This skill also allows to backup and restore the Test Repository structure in Xray, which can be useful for maintaining the organization of tests in Xray and ensuring that the Test Repository structure is preserved during migrations or other changes. The instructions for backing up and restoring the Test Repository structure are also described in the sections below.

## Flow

The user must specify the action he wants to perform, whether it's organizing the Test Repository of a given project, organizing a Test Plan, backing up the Test Repository structure, or restoring the Test Repository structure for a given project. Based on the specified action, follow the instructions described in the sections below to perform the desired folder management task in Xray. Always follow the instructions for each specific task to ensure a consistent and accurate management of folders in Xray.
